The STM8S103, R421A08, R221A04 and R413D08 are relay boards that are controlled from your computer, through the USB port. They allow you to control any 12 volt appliance such as led lights for example. You can install an additional 120/220 volt relay that is 12 volt activated. The relay boards can also process input signals such as switches and sensors. Contains MS Visual studio C++ source code. Windows 7/8 compatible.
